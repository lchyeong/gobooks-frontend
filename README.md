# Gobooks - 느리지만 꾸준한 독서의 즐거움, 인터넷 책 쇼핑몰

**Gobooks**는 천천히, 꾸준히 읽는 독서의 즐거움을 강조하는 인터넷 책 쇼핑몰입니다.  
거북이를 모티브로 하여 독자들에게 책을 구매하고 끝까지 읽어낼 수 있다는 희망을 전합니다.

---

## 🌟 서비스 목표

1. **통합 도서 구매 쇼핑몰**
   - Gobooks는 책 구매와 독서 관리를 위한 원스톱 플랫폼으로 독자들에게 최적의 환경을 제공합니다.

---

## 🚀 도전 목표

1. **통합 도서 구매 쇼핑몰 CRUD 구현**
   - Spring Boot와 React를 활용하여 책 데이터, 사용자 상호작용, 구매 워크플로우를 효율적으로 관리하는 시스템을 구축했습니다.

---

## 🔮 향후 목표

1. **React Error Boundaries로 에러 처리 강화**
   - 예기치 못한 에러를 사용자에게 자연스럽게 처리하여 더 나은 사용자 경험 제공.
2. **카카오 API를 활용한 도서 정보 배치 처리**
   - 카카오 API를 통해 도서 데이터를 동적으로 업데이트하고 풍성하게 만들기.
3. **검색 엔진 최적화(SEO)**
   - 플랫폼을 검색 엔진에서 더 잘 노출되도록 최적화.
4. **Jenkins로 CI/CD 구현**
   - 자동화된 배포 파이프라인을 통해 안정적이고 효율적인 개발 환경 조성.
5. **Docker로 프로젝트 빌드 관리**
   - Docker를 활용하여 개발, 테스트, 배포 환경의 일관성을 확보.
6. **E2E 테스트 추가**
   - 통합 테스트로 애플리케이션 신뢰도 강화.
